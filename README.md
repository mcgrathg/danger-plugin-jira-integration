> [Danger](https://github.com/danger/danger-js) plugin to integrate your pull request with JIRA

## Usage

Install:

```sh
yarn add danger-plugin-jira-integration --dev
```

At a glance:

```js
// dangerfile.js
import jiraIntegration from 'danger-plugin-jira-integration'

jiraIntegration()
```
